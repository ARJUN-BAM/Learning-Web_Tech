'position:' Property

About position property & Type of position properties
  • static defualt position property which allows elements to not overlap with each other.
  • fixed to fixed an element into your viewport and scrolling won't have any effect onto the element.
  • sticky seems like a preety useful property. used for an element to stick onto a certain position into the viewport.
  • Other type of display property which are Relative[ recommended ]:
    • absolute here element will take the reference for positon from its nearest parent position.
    • relative here element will take the reference for positon from its nearest parent position.
    • Note: the element with relative position will take the reference from its original position. where as absolute position will take reference from the html tag or viewport.
    • Note: the element with relative position will not loos its position where in absolute position element will loose its position.
