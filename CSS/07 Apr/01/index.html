Display Property

Different Attribute in Display
  • inline used to change a block element property into a inline element. Ex: display:inline;
  • block used to change a inline element property into a block element. Ex: display:block;
  • inline-block used for giving both block and inline elements property to any elements.
  • flex property is used to make all the first generation child element into inline_block element, Creating a flexible box which is parent element.
  • none used for what?
  • Note: justify-content
  • Note: align-items
  • Note: overflow property to create a over-sized content into a hide and show through scroll. use auto
  • Note: flex-wrap is another property which creates a grid like layout.``
