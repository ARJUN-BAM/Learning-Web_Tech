Simple Selectors
  • Tag name is also a selector used to target element from same tag name.
  • id attribute is a selector used to target a single element. You can use id for multiple element but that will surely create problems.
  • class attribute is a selector used to target a single or multiple elements.
  • Grouping is also a way to select multiple targets. Example: div , #id , .class{ }
  • * is a selector used to target all the elements present into the document.
  • id >> class >> tagName >> * : Grouping [ as per the order of selector and codeline ]
  • You can use the !important on the value of the property to give it the highest priority. Example : " background-color: green !important; "
