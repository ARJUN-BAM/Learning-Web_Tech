Media Query

Why media query?
  • This property is used to create a webpage more responsible to deffernt viewport size.
  • We can include new css properties to the elements when screen size is different.
  • We can add include style changes for these elements who will also includes the original css property.
  • @media (max-width : 1000px) { section>main{ background-color: blue; display: flex; justify-content: space-between; } } where max-width which is 1000pc is breakpoint.
  • Note: Breakpoints are: 1920px,
