Pseudo Element Selector

Pseudo_element_selector
  • first-letter for trageting the first letter of a content.
    Example: p::first-Letter{ }
  • first-line for trageting the first line of a content.
    Example: p::first-line{ }
  • selection this selector is used to change some property [ like text-color and background-color ] onto the selected content.
    Example: p::selection { }
  • marker is used to change the bullets signs into a list.
    Example: li::marker{ content: "" }
  • before is used to target a tag and to add some more content( using content attribute) and property into the added content before the targeted content.
    Example: p::after{ }
  • after Vice Versa For after.
    Example: p::after{ }.
  • Note:before and after is generally used in animations. Important Property is content which is used to add content into the tag.
