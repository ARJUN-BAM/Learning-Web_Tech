Combinator Selectors [ Working with relation ]
  • General Sibling Selector is used to target the next tag inside a same element making the tag as sibling [ like blood related sibling ]. Example : Div~P {}
  • Adjecent Sibling selector used for targeting the adjacent exactly next tag [ it's Exactly next ]. Example : Div+p {}
  • Child Selectortargets adjacent child tag consider having father-son relation being 1st generation child. Example: div>p{ }.
  • Descendent selector used to targets the all child tag from any generation.Example: div p{ }
  • Note: The siblings are only consider when counting down not counting up.
