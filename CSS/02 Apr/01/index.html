Attribute Selector

  • Can use through [attribute]
  • Use [attribute="Entire_value"]
  • As any element have different attribute and some element also have some specific attribute thats why we us this selector to tarfet those specific element.
  • Use[attribute ^="char"]for target the attribute which have a value starting in original its attribute's values.
  • Use[attribute $="char"]for target the attribute which have a value ending in its original attribute's values.
  • Use[attribute |="char"]for target the element whose attribute value have is sperated by hyfun or not '-' .
  • Use[attribute ~="char"]for target the element whose attribute value have is sperated by space or not ' '.
  • Use [attribute *="char"] for target the element whose attribute value is somewhere present in the value mentioned. universal selector.
