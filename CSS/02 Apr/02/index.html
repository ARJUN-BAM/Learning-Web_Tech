Units into the CSS

Units Types
  • Absolute Unit'
    • Note: these units are the fixed unit which will not be changed and remains contants in any case.
    • px (Pixel)
    • cm (centimeter)
    • mm (milimeter)
    • in (inch)
    • pc (same as pixel)
    • pt (same as pixel)
  • Relative Unit
    • Note: These units changes acorrding to the enviroment or platform, It has to take refrence with something. these units takes reference from another element like root element,viewport and parent element.
    • % (Percentage)
    • vh (viewport height)
    • vw (viewport width)
    • em (Element) will specify how much times the value will be relative to the parent value. it will take refrence from the parent element.
    • rem (Root Element) will specify how much times the value will be relative to the Root element's value which is html tag. it will take refrence from the root element.
    • vmax between width and height which ever size is maximum it will take refrence from that.
    • vmax between width and height which ever size is minimum it will take refrence from that.
    • Note:rkr,mThe font size in the root element is 16px which is also a default for p and container.
    • Note: Dont let the first generation child tag of body's size be defined by using percentage%.
    • Note: We can also use % for specifying the font size but its not recommendated as it will be complex.
    • Note: Default font size is 16px.
