Font Properties

Properties used with font
  • text-align can be used to align the text into the block or where to start the text.
  • text-transform can change the text into uppercase and lowercase.
  • text-shadow can be used to put shadow into the text.first value is left or right and second value is top or bottom and thrid value is how much shadow and last value is the color. Example: text-shadow: -15px 5px 3px black;
  • text-indent used to give the indentation to the starting of the paragraph.
  • text-decoration used for giving three type of line to the content. can also add multiple properties to the decoration. Example: text-decoration: underline double 2px red; which are text_dec_type, text_dec_style, text_thick_weight and text-deco-color.
  • text-decoration-style
  • text-decoration-thickness
  • text-decoration-color
  • text-decoration-line used to add and specify what type of line to the content. Example: text-decoration-line: underline; which are text_dec_type, text_dec_style, text_thick_weight and text-deco-color.
  • Note: Properties which can have more than one value are called short_end_property.
  • Note: text-decoration is a short-hand-property.
