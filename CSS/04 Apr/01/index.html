Colors

Type of color we use in css
  • Predefined colors used in general like red, blue and so on.
  • RGB Stands for Red , Green and Blue it uses 0-255 values for these colors. Example: rgb(0,0,0) which will give black color.
  • HexaDecimal_Colorcode [ #ffffff ] this value is same as the rgb color where each pair represent red, green and blue.
  • HSL hue has a color wheel with 0 to 360 degree for getting color where 0or260 for red, 120 for green and 240 for blue. S stands for saturation and B stands for Brightness. Example: hsl(240deg, 100%, 45%); which is hsl(heu,saturation,brightness).
  • Note : rgb and hsl also have another version which is rgba and hsla where a is known as alpha which has a range of 0-1 which shows the transparency of the color.
  • Note : We can use these colors in any containers or any element in html using the css.
