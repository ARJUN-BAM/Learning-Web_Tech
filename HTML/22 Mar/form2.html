Attributes

readonly is an attribute used for making a input only for read so that user can't edit the value.

Disabled is also an attribute used for disabling any input tag meaning becoming readonly and little transparent.

placeholder used to emiting a value for indication what to enter into the input tag.

Creating a input field for Mobile Number

minlength & maxlength are the input text tag attribute used for specifying how much length of characters should be entered by a User.
